[h1]Overview[/h1]
Do you like terraform stations from my mod [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2622411084]Aesthetic Terraform Stations[/url]? Do you also want to have properly-skinned terraform stations for [url=https://steamcommunity.com/sharedfiles/filedetails/?id=864541681]We are the Imperial Navy[/url]? Then this mod is for you!

[h1]Changes[/h1]
Adds a terraform station definition with an active terraforming beam for graphical culture [b]wain_01[/b], required for Aesthetic Terraform Stations to apply the correct appearance.

[h2]Compatibility[/h2]
Should work with practically everything that also works with We are the Imperial Navy and Aesthetic Terraform Stations.

Built for Stellaris version 3.7 "Canis Minor." Not compatible with achievements, but neither are the dependencies.

[h3]Required Dependency Mods[/h3]
[list]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=2622411084]Aesthetic Terraform Stations[/url] enables the very old-school terraform stations as visual markers for terraforming planets
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=864541681]We are the Imperial Navy[/url] for the original graphics and other ship-related code
[/list]
